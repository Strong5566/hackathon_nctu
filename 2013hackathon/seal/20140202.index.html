活動介紹

什麼是黑客松

「Hackathon = Hack + Marathon。」Hack原意是駭客，泛指對系統的操作方法、構造和運作原理非常瞭解的人，之後衍申為動詞，意指因應自身需求改造系統，或者乾脆從零開始打造。Marathon應該不用多解釋了吧！就是馬拉松的意思。而「Hackathon」這個組合字，顧名思義，就像是工程界的馬拉松：集合一群志同道合的朋友，以過人的毅力和耐力，在有限的時間內，近乎不眠不休地日夜趕工，將腦海中的構想或專案實現。在國外舉辦的Hackathon，時間可能長達數日至一週不等。不過後來，也出現一些Hackathon，因應不同的環境及需求，把活動時程壓縮在1天之內結束。參與團隊可以實驗性地先開發出系統的prototype(原型)，發表之後，參酌使用者的意見，再決定是否要繼續完成細節，或者是要修改原先的設計。

更多有關Hackathon的介紹，可以參考Wikipedia的解說

關於交大黑客松

這次的「交大黑客松」為期兩天一夜，希望藉由這個活動，促成同樣對系統開發有興趣的伙伴互相認識。除此之外，本次活動與其他Hackathon有個很不一樣的地方：以往Hackathon的開發專案是由與會的個人所自行發想決定，若是企業有參與，扮演的角色也只是從旁支援活動流程，而不干涉開發專案本身；然而，本次活動的題目皆是由目前活躍於產業界的企業所提出，因此題目本身將更貼近當前的業界需求。參加本次活動的朋友可以發揮所長，幫助企業解決問題，同時也能和企業面對面交流想法。於活動尾聲的成果發表，更邀請企業和所有參加者一起針對各團隊的作品評筆，並給予意見回饋。假若作品受到企業所青睞，那麼開發團隊還能夠獲得企業所贊助的獨家獎賞。

獎勵辦法

出題企業將會提供獎金與獎品鼓勵有所貢獻的開發團隊，甚至直接提供至該企業工作或實習的機會。
原則上，出題企業將依照下列評斷標準，頒予開發團隊全部或者部份比例的獎賞：

  1. 作品完成多少項企業所提出的需求
  2. 作品對企業的貢獻程度
  3. 功能實作的難易度
  4. 作品的創意性

除此之外，只要完整參與兩天一夜的活動，一律授予交大黑客松專屬紀念品和活動參與證書！

另外，為鼓勵學生人才，主辦單位將據依當天參與者的整體表現，選出幾位表現優秀學生 (視報名人數情況調整)額外頒授「傑出表現獎」。 除此之外，由學生組成之兩人以上團隊，主辦單位亦將甄選出表現優秀的隊伍另外頒予獎勵。

報名流程

報名程序分為兩部份：「網路報名」和，「繳費」。
請注意！只完成「網路報名」但未「繳費」視同未完成報名程序，自動放棄參加資格，原名額將依序釋出給備取的參加者。

  1. 網站報名：填寫基本資料、選填題目類型志願，02/02 截止。網路報名成功後，大會將寄發確認信。
  2. 02/03 大會公告「網路報名」成功，已錄取的參賽隊伍
  3. 02/08 第一階段報名成功隊伍繳費期限
  4. 02/09 大會發出候補通知
  5. 02/12 候補報名成功隊伍繳費期限
  6. 02/22 Hackathon當天：公佈確切題目並實作。

報名須知

題目共分成四大類，在此列出所需要的技術種類：

  1. 手機應用 - iOS/Android/RWD
  2. Web前後端 - 前端：HTML/CSS/JavaScript，後端：PHP/MySQL ...
  3. 軟硬整合 - C/C++/Python/Java...
  4. 資料分析 - 關聯式資料庫及任何語言

請於網路報名時依照志願序填寫題目類型，並且儘量在專長中說明團隊或個人所具備的技術或其他能力。完成後主辦單位將會分配適當的題目給各個團隊。

  1. 團體報名者：每隊人數2~6人，團隊中至少需有一人會撰寫題目類型所需的程式語言。
  2. 個人報名者：請務必明確填寫個人的專長，如果能夠再「備註」欄位自我介紹更好，以利於主辦單位幫忙媒合適當的合作伙伴。

強烈建議攜帶筆記型電腦。活動當天提供電源和無線網路，若要使用主辦單位提供的電腦，必須移駕至公用區，但公用區內不開放團隊討論或飲食。

問題集

  • 活動當天在哪住宿？

    Hackathon理論上是不會停下來睡覺的喔！不過，如果覺得太累想要休息，也可以移動至休息區，趴在桌上小睡，或者自己帶睡袋來打地舖。

  • 活動進行中可以暫時離開會場嗎？

    可以。這不是「入圍」，除了開場和閉幕，如果有需要，可以自行暫時離開，但要記得回來！

  • 活動內容具體來說是什麼?

    這是秘密喔！快組隊報名來體驗一下就知道了。可以確定的是：來參加之後，一定會更認識自己、技術更精進、認識不少新朋友，不虛此行！

  • 非來不可的理由?

    • 與企業交流，直接認識業界的想法和需求，進而瞭解如何學以致用。
    • 幫助企業解決問題，培養學校裡最欠缺的實作能力。成功達成任務的話，還可以獲得企業的獨家好禮。
    • 認識與你同樣積極，而且理念相近的伙伴，歷經兩天一夜的相處，凝聚出深厚的戰友革命情感。
    • 體驗微型創業的經驗，培養團隊合作能力，就算未來沒有選擇創業，也是職場上炙手可熱的人才。
    • 在交大，有好吃的點心，高級的活動會場，這樣的工作環境多難得啊！怎麼可以白白放過呢？

  • 參加這個活動，需要什麼樣的能力?

    如果您是自行組成團隊報名，則團隊中至少需有一人會撰寫出題企業指定的程式語言。報名時會提供選項，讓您勾選團隊所使用的程式語言，我們將會依據您勾選的的結果分配適合的題目，大會將再次寄信通知各團隊所需具備的程式語言能力。

    而若您是個人報名的，只要您有一顆熱忱的心以及滿滿創意，則沒有一定要會撰寫程式的限制，我們會幫您引介適合的合作伙伴。

  • 我要怎麼知道我報名成功了呢?

    報名流程分成「網路報名」和「繳費」兩個階段。 大會收到您在網路上填寫的報名資料之後，會寄信通知您已完成「網路報名」！ 請注意：完成「網路報名」不代表已報名成功！必須完成「繳費」之後才算是報名已成功！

    若您在匯完報名費後發現不克參與，請寄信聯絡工作人員!
